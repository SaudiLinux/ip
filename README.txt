# أداة جمع معلومات أرقام الهواتف

أداة بسيطة لجمع وتحليل معلومات أرقام الهواتف، تم تطويرها بواسطة Saudi Linux.

## المتطلبات الأساسية

- بايثون 3.6 أو أحدث
- مكتبات بايثون المطلوبة (مذكورة في ملف requirements.txt)

## التثبيت

### الطريقة التلقائية (مستخدمي ويندوز)

1. قم بتشغيل ملف `setup_venv.bat` بالنقر المزدوج عليه.
2. سيقوم الملف تلقائيًا بإنشاء بيئة افتراضية وتثبيت جميع المتطلبات.

### الطريقة اليدوية

1. قم بإنشاء بيئة افتراضية:
   ```
   python -m venv venv
   ```

2. قم بتفعيل البيئة الافتراضية:
   - في ويندوز:
     ```
     venv\Scripts\activate
     ```
   - في لينكس/ماك:
     ```
     source venv/bin/activate
     ```

3. قم بتثبيت المتطلبات:
   ```
   pip install -r requirements.txt
   ```

## كيفية الاستخدام

### واجهة سطر الأوامر

1. في ويندوز، قم بتشغيل ملف `run_tool.bat` بالنقر المزدوج عليه.
2. أو قم بتشغيل الأداة مباشرة من سطر الأوامر:
   ```
   python phone_info_tool.py +966501234567
   ```

### الواجهة الرسومية

1. في ويندوز، قم بتشغيل ملف `run_gui.bat` بالنقر المزدوج عليه.
2. أو قم بتشغيل الواجهة الرسومية مباشرة من سطر الأوامر:
   ```
   python gui_tool.py
   ```

3. أدخل رقم الهاتف في الحقل المخصص.
4. انقر على زر "تحليل الرقم".
5. ستظهر النتائج في منطقة النتائج.
6. يمكنك تصدير النتائج بتنسيقات مختلفة (JSON، CSV، Excel، HTML).

## الميزات

- التحقق من صحة أرقام الهواتف
- الحصول على معلومات أساسية (رمز الدولة، اسم الدولة، المشغل)
- الحصول على معلومات الموقع الجغرافي
- الحصول على معلومات المنطقة الزمنية
- تصدير النتائج بتنسيقات متعددة
- واجهة سطر أوامر وواجهة رسومية
